# Project Handoff — 2026-07-01

## What This Project Does

Automated pipeline for counting vehicles at the Woodlands Causeway (Singapore) using LTA traffic cameras. Images are scraped every 5 minutes, auto-labelled by RT-DETR-X, manually curated, then used to fine-tune a compact vehicle detector that runs natively on Apple Silicon (MLX).

**Cameras:** 2701 (Causeway towards Johor), 2702 (Checkpoint towards BKE), 2704 (Flyover towards Checkpoint)  
**Vehicle classes:** motorcycle (0), car (1), bus (2), truck (3), train (4)  
**Stack:** Python 3.14, Dagster, Ultralytics, MLX, Streamlit, SQLite

### JS Patch (not committed — lives in `.venv` only)

**File:** `.venv/lib/python3.14/site-packages/streamlit_image_annotation/Detection/frontend/build/static/js/main.f3db844f.js`

Added Option+D (`event.key==="d" && event.altKey`) keydown handler that deletes the currently selected bounding box. The handler filters `m` (rectangles) by `S` (selectedId) and calls `b` (setRectangles) + `x` (setSelectedId=null). Dependency array extended from `[m]` to `[m,S]`.

**Must re-apply if venv is recreated.** Patch script:
```python
JS = ".venv/lib/python3.14/site-packages/streamlit_image_annotation/Detection/frontend/build/static/js/main.f3db844f.js"

       'label_id:l.indexOf(e.label),label:e.label}}));jr.setComponentValue(t)}'
       '};return window.addEventListener("keydown",e),function(){window.removeEventListener("keydown",e)}}),[m])')
new = ('if(f&&" "===e.key){var t=m.map((function(e,t){return{bbox:[e.x,e.y,e.width,e.height],'
       'label_id:l.indexOf(e.label),label:e.label}}));jr.setComponentValue(t)}'
```
